# Legal notes

This repository does not include Tree of Savior client binaries, IPF assets,
patch files, videos, DLLs, or other proprietary game files.

The installer copies those files from an existing local Steam installation of
Tree of Savior and applies CloverTOS configuration files locally.

Do not commit the generated `C:\CloverTOS-Local` client directory, zip archives
containing the game client, or copied Steam game files to this repository.

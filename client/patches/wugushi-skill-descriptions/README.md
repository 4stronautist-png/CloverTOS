# Wugushi skill descriptions

Updates Wugushi client text to match the Laima package handlers:

- Zhendu
- Latent Venom
- Poison Pot
- Golden Frog
- Wide Miasma
- Crescendo Bane
- Zhendu attributes and arts

Run:

```powershell
.\Apply-WugushiSkillDescriptionPatch.ps1 -ClientPath "C:\CloverTOS-Local"
```

Close and reopen the game client after applying the patch so the language data is reloaded.

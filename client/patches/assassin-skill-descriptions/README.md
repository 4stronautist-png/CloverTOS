# Assassin Skill Descriptions Patch

This patch applies the Assassin client tooltip updates to a local CloverTOS client.

```powershell
powershell -ExecutionPolicy Bypass -File .\client\patches\assassin-skill-descriptions\Apply-AssassinSkillDescriptionPatch.ps1 -ClientPath "C:\CloverTOS-Local"
```

The patch updates the English and Portuguese `SKILL.tsv` entries used by Hallucination Smoke, Commission to Kill, and Annihilation: High Speed.
